Splitter

since: 1.3.0

split_with config allows us to split the backup file into multiple files, by each chunk size limit.

Config

  • chunk_size - Each chunk file size limit

Example

models:
  my_backup:
    encrypt_with:
      type: openssl
      password: 123456
      salt: false
    split_with:
      chunk_size: 100m
    storages:
      local:
        type: local
        keep: 2
        path: /tmp/demo
