Feishu (飞书) - Notifier

since: 1.5.0

Feishu is a team communication tool.

We can send Feishu message by use Webhooks, you can follow the Feishu Incoming Webhooks Guide to get the webhook url.

Configuration

models:
  my_app:
    storages:
      local:
        type: local
        keep: 10
    notifiers:
      feishu:
        type: feishu
        url: https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxx
        on_success: false
